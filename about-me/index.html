Linux n' That

I really like Linux. I started daily driving Linux Mint when I was 16 and it's been a fun journey ever since. I've been distro-hopping a lot since so I won't list what distros I'm on currently to avoid dating this section but some of my favorites include CachyOS (Arch Derivative), NixOS, and Linux Mint. There are tons of reasons to get into Linux that you can find online so if you haven't and are interested in software at all, I recommend looking into it.

Free and Open-Source
